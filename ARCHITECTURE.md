# Architecture

## Agent Roles
- **Brain (Planning):** Responsible for analyzing tasks and deciding the workflow.
- **Hands (Execution):** Responsible for executing specific coding or deployment tasks.

## Slack Channel Scheme
- We use a dedicated Slack channel (`#new-channel`) for bot communication and logging.

## Model Routing
- The system uses Hermes configuration to route tasks between different AI models (like Gemini and others).
