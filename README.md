# Forge 2 Project

## About
This is a Kanban board application integrated with Slack, featuring autonomous agents for task management.

## Run Instructions
1. Clone the repository.
2. Set up your `.env` file with necessary API keys.
3. Install dependencies and run the backend/frontend servers.
